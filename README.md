# SBF Lern-App — Sportbootführerschein Binnen & See

Interaktive Lern-App für den Sportbootführerschein (SBF) mit Quiz-Modus und KI-Chat.

## Features

- Quiz-Fragen für SBF Binnen und SBF See
- KI-gestützter Chat für Navigationsfragen
- Fortschritts-Tracking (LocalStorage)
- Responsive Design (Mobile + Desktop)
- Standalone HTML — keine Dependencies

## Starten

Einfach `index.html` im Browser öffnen. Kein Build-Schritt nötig.

## Herkunft
